это учебный проект 

веб приложение Django - для внутреннего использования в зоомагазине

автоматизация учёта товаров - можно отслеживать:
сколько сколько осталось на складе
сколько продано 
сколько возвратов
финансовый отчёт (считает на какую сумму продали - на какую сумму вернули)


есть отдельная аунтификация для сотрудника и руководителя(может просматривать фин.отчёт)

управление складом:
    каталог с категориями,ценами и остатками
    теги для класификации
    описание товаров

транзакции:
    форма для продаж и возвратов
